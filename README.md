# 動きの概要

毎時01分に全プレイヤーの整地を取得->player_data{today}.jsonに記録  
23:58に全プレイヤーの整地量を取得->player_data{today}.jsonに記録  
再度ランキングページにアクセスし、上位40人の整地量を取得->player_data{today}.jsonをもとに上位40人の辞書を作成  
それもとにグラフ作成・保存
